극마계촌은 전설적인 마계촌 시리즈를 PSP로 부활시킨 횡스크롤 액션 게임입니다. 플레이어는 기사 아서가 되어 마왕에게 납치된 공주를 구하기 위해 수많은 몬스터와 함정이 가득한 스테이지를 돌파하게 됩니다. 화려해진 그래픽과 다양한 무기, 마법, 방어구 시스템을 도입하면서도 시리즈 특유의 높은 난이도와 긴장감 넘치는 액션을 그대로 계승해 도전 욕구를 자극하는 작품입니다

작업 후기

어릴적 추억이 생각나 작업을 진행한 게임입니다.

해당 게임은 전용 폰트를 쓰지 않고 독자적인 폰트 테이블을 사용하기에 우선적으로 폰트테이블을 찾는 부분이 어려웠습니다.

전체적인 대사의 내용은 많지않아 폰트 테이블 확인 후 작업은 빠르게 진행하였으며 전체적인 플레이 검수는 확인되었으며 문제점은 발견되지 않았습니다.

하지만,난이도가 상당히 있는 게임이라 실제 조작중 화남을 주의하세요.
